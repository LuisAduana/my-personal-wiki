Repositorios Rotos

Por alguna razón los repositorios donde Manjaro descarga sus paquetes se rompen.

Para solucionar esto se deben actualizar las bases de datos de claves PGP de los desarrolladores de Arch Linux y Manjaro.

sudo pacman-key --refresh-keys

Después se cargarn las nuevas claves en pacman.

sudo pacman-key --populate archlinux manjaro

Eliminamos los paquetes que se intentaron actualizar y marcaron error.

sudo pacman -Scc


¡Importante! reiniciar el sistema.

Finalmente, sincronizar las bases de datos de los paquetes con el siguiente comando.

sudo pacman -Syy


Si no funciona. Primero nos aseguramos que el primer mirror esté en la lista blanca. Despúes

sudo rm -R /var/lib/pacman/sync

Repetir todos los pasos anteriores.
